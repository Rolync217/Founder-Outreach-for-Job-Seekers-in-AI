# Outreach Style (optional)

This file is loaded by the drafting node if present. Use it to layer personal
style preferences on top of the base outreach rules in `skills/outreach-rules/`.

Leave this file empty or delete it entirely to use only the base rules.

## Example additions

- Always open with a specific observation about what the company is building
- Never use the word "synergy"
- Prefer short sentences under 15 words
- Sign off with your first name only
